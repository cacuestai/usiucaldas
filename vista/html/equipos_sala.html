Ayudas

  1. Campo descripcion:al momento de adicionar o modificar registros, en este campo puede comentar una breve descripcion del equipo al que se hace referencia
  2. Campo partes equipo:este campo sirve para comentar las partes principales por las que esta compuesta el equipo.
  3. Campo software equipo:en este campo puede escribir los principales programas que se encuentras en el equipo al que se hce referencia.
